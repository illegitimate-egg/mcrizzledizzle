# Logging

These functions all go directly to the server logs, as you'd expect. They all take a message and handle the rest for you.

## info
Signature:
```rust
fn info(msg: String)
```

## warn
Signature:
```rust
fn warn(msg: String)
```

## error
Signature:
```rust
fn error(msg: String)
```

## debug
Signature:
```rust
fn debug(msg: String)
```
